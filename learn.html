What is ARIA?

ARIA stands for Accessible Rich Internet Applications. ARIA makes advanced web applications accessible and usable for people with disabilities.

Why use ARIA?

By incorporating ARIA into HTML code, your web application will be accessible to people with disabilities such as visual impairment that prevent them from reading what is displayed on a user interface.

How does this work?

A person who is visually impaired uses software programs called screen readers.
ARIA attributes in a program signal to a screen reader what text and functionality displayed on a screen needs to be communicated to the visually impaired user.
The screen reader reads aloud the text and functionality to the user.

What does our ARIA generator do for you?

After inputting your HTML code into our input box, we will parse through your code adding ARIA attributes where necessary, producing an output with all required ARIA attributes that will allow people with disabilities to access your web application.
